The Editor in the Zyra Ecosystem

From CLI manifest to visual graph to executable pipeline — a seamless round-trip.

Zyra CLI

→
Manifest JSON
Stages, args, ports, types
→
Visual Editor
Drag, connect, configure
→
Pipeline YAML
Topologically sorted output
→
Execute
Run with Zyra CLI or via the editor

The editor is Layer 4 of the Zyra stack — a visual interface that makes CLI pipelines accessible to researchers and domain experts who prefer graphical tools over command lines. It doesn't replace the CLI; it complements it. Every graph you build in the editor exports to the same YAML that power users write by hand.
