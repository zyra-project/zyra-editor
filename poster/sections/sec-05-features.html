Capabilities

Key Features

Everything you need to visually build, configure, execute, and export scientific data pipelines.

Drag-and-Drop Pipeline Building

Browse stages in the Node Palette, drag them onto the canvas, and connect type-safe ports. The @zyra/core library validates compatibility — only matching types can link.

Live Argument Editing

Select any node to configure its CLI arguments via the detail panel. Supports text, numbers, booleans, file paths, dropdowns, and cron expressions.

Real-Time Execution & History

Run pipelines with WebSocket log streaming and per-node status indicators. Track past runs with timing visualization and Gantt charts.

Pipeline YAML Round-Trip

Export your visual graph to pipeline.yaml via topological sort, or import existing YAML back into the editor. Full round-trip between visual design and CLI.

AI-Powered Planning

Describe what you want in natural language — the Planner Panel generates a pipeline plan, translates it to a visual graph, and lets you refine it.

Security & Resource Management

Secrets are AES-256-GCM encrypted at rest — never written to YAML. Define reusable resources (URLs, paths, credentials) and reference them across nodes.
